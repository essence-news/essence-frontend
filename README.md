# essence-frontend


## Setup

**Use latest version of node 21+ and npm 10+**

Run these commands

`npm install`

**To run locally**

`npm start`
and once its running, press `w` for web version

**To build for web**

`npm run build:web`

**To build for APK**

`npm run build:apk`

You will need EAS credentials for this.

**To preview build**

`npx serve build`
